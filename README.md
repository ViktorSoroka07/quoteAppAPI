### Getting started

#### Install dependencies:

```
npm i
```

or

```
yarn i
```

#### Run development server:

```
npm run start:dev:pm2
```

#### Run production server:

```
npm run start:env
```

or

```
npm run start:env:pm2
```

#### Run unit tests:

```
npm test
```
